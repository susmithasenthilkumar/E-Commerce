## 📊 E-Commerce Customer Behavior Analysis and Sales Prediction System Using Python

---

## 📄 Project Description

This project analyzes customer behavior in an e-commerce environment and predicts purchasing patterns using data science techniques. It simulates real-world data including customer activity, product pricing, purchase quantity, and transactions. The system applies preprocessing, statistical analysis, machine learning, and visualization to generate meaningful insights.

---

## 🔍 Features & Analysis

### ✅ Revenue Calculation

Revenue is calculated using:

```
Revenue = Quantity × Price
```

### ✅ Customer Behavior Analysis

* Analyzes customer activity and purchase patterns
* Identifies trends in buying behavior

### ✅ Correlation Analysis

* Finds relationship between:

  * Customer activity
  * Product category
  * Revenue

### ✅ Machine Learning

* Logistic Regression → Predict purchase decision
* Linear Regression → Forecast sales trends
